# Frontend (React)

## Requisitos

- Node.js

## Configuração

Crie um arquivo `.env` ou `.env.local` com:

```
VITE_API_BASE_URL=http://localhost:8787
```

## Rodar localmente

```bash
cd /Users/satin/Work/bolao-brasileirao/apps/web
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (Cloudflare Pages)

- Build command: `npm run build`
- Output directory: `dist`
